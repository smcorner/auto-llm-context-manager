================================================================================
PART 1: WHAT IS THIS APP?
================================================================================

AI Context Manager Pro solves a big problem:

PROBLEM: AI chatbots (ChatGPT, Claude, etc.) forget everything after each 
         conversation. You keep repeating the same information!

SOLUTION: This app:
  * Saves your conversations automatically
  * Extracts important information (projects, tasks, dates, budgets)
  * Generates smart prompts with FULL CONTEXT
  * Organizes everything by PROJECT/WORKSPACE
  * Shows you what AI "remembers" and "thinks"

================================================================================
PART 2: MAIN FEATURES
================================================================================

MULTI-WORKSPACE SUPPORT
   - Create separate workspaces for each project
   - Each workspace has its own memory
   - Switch between projects easily
   - Export/Import workspaces

MEMORY BANK
   - Stores all your conversations
   - Remembers projects, technologies, names, numbers
   - Persists across browser sessions

AI THINKING LOG
   - Shows step-by-step processing
   - See what memory is accessed
   - Watch extraction happen in real-time
   - View AI insights being generated

AUTO AGENDA
   - Automatically creates tasks from conversations
   - Priority levels (High, Medium, Low)
   - Mark tasks as In-Progress or Completed

AI INSIGHTS
   - Observations about your work patterns
   - Recommendations for improvement
   - Warnings about constraints (deadlines, budgets)
   - Connections between technologies

ENHANCED PROMPT OUTPUT
   - Copy-paste ready prompt for AI
   - Includes full context from memory
   - Previous sessions summary
   - Current agenda items

================================================================================
PART 3: HOW TO USE (STEP BY STEP)
================================================================================

STEP 1: CREATE A WORKSPACE
--------------------------
1. Look at the left sidebar (click menu icon on mobile)
2. Click "New Workspace"
3. Pick an icon (example: mobile icon for mobile app project)
4. Pick a color (helps identify visually)
5. Enter a name: "My Mobile App"
6. Enter description: "React Native food delivery app"
7. Click "Create"

STEP 2: ADD A CONVERSATION
--------------------------
1. Make sure correct workspace is selected
2. Find the "Input Conversation" box
3. Paste your conversation (see examples below)
4. Click the big "PROCESS WITH AI" button

STEP 3: WATCH THE PROCESSING
----------------------------
The AI Processing Log shows 8 steps:
  Step 1: Accessing Memory Bank
  Step 2: Parsing Conversation  
  Step 3: Extracting Information
  Step 4: AI Reasoning Process
  Step 5: Generating Insights
  Step 6: Updating Agenda
  Step 7: Synthesizing Context
  Step 8: Generating Enhanced Prompt

STEP 4: GET YOUR ENHANCED PROMPT
--------------------------------
1. Click "Enhanced Output" tab
2. See the full context prompt generated
3. Click "Copy" button
4. Paste into ChatGPT/Claude/other AI
5. Your AI now has FULL CONTEXT!

STEP 5: CONTINUE ADDING CONVERSATIONS
-------------------------------------
1. After each AI chat session, paste the conversation
2. Memory keeps growing with more context
3. Enhanced prompts get smarter over time
